3. The Product Class

We need to define what a "Product" actually is.

Action: Create Class

  • Right-click Package → New → Java Class.
  • Name: Product.

Action: The Code

public class Product {
    
    private String name;
    private double price;

    public Product(String name, double price) {
        this.name = name;
        this.price = price;
    }

    public String getName() { return name; }
    public double getPrice() { return price; }

    @Override
    public String toString() {
        return name + " ($" + price + ")";
    }
}

We need a method that runs every time we open the window to check for new products.

In ShopFrame source:

public void updateList() {
    DefaultListModel model = new DefaultListModel();
    
    // Loop through the global inventory
    for (Product p : JohnDoeShop.inventory) {
        model.addElement(p);
    }
    
    lstProducts.setModel(model);
}

8. The Order Class

Before we can sell things, we need an object to represent a "Sale".

Action: Create Class

  • New Java Class → Order.

Action: The Code

import java.util.ArrayList;

public class Order {
    
    // An order is a list of products
    private ArrayList<Product> cart;
    private double total;

    public Order() {
        cart = new ArrayList<>();
        total = 0;
    }

    public void addToOrder(Product p) {
        cart.add(p);
        total = total + p.getPrice();
    }

    @Override
    public String toString() {
        return "Items: " + cart.size() + " - Total: $" + total;
    }
}
